## Ideia 💡:
Desenvolver um identificador de faixa etária utilizando **javascript**.

## Funcionalidades:
- Campo de seleção ano de nascimento;
- Seleção de sexo;
- Validações;
- Ilustrações representativas;
- Mensagem de erro ao informar dados inválidos.
